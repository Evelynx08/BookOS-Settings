📦 BookOS APT

Debian/Ubuntu package repository for BookOS Settings

stable amd64 GPG signed

🚀 Installation

Add the repository to your system in two steps:

1. Import the GPG signing key

sudo install -d -m 0755 /etc/apt/keyrings
curl -fsSL https://evelynx08.github.io/BookOS-Settings/KEY.gpg | sudo tee /etc/apt/keyrings/bookos.asc > /dev/null

2. Add the repository source

echo "deb [signed-by=/etc/apt/keyrings/bookos.asc] https://evelynx08.github.io/BookOS-Settings bookworm main" | sudo tee /etc/apt/sources.list.d/bookos.list

3. Install BookOS Settings

sudo apt update
sudo apt install bookos-settings
💡 The bookworm codename works on Debian 12+ and Ubuntu 22.04+. The package depends on libwebkit2gtk-4.1-0 and libgtk-3-0.

🔄 Updating

New versions are picked up automatically through your normal system updates:

sudo apt update
sudo apt upgrade

🗑️ Removal

sudo apt remove bookos-settings
sudo rm /etc/apt/sources.list.d/bookos.list
sudo rm /etc/apt/keyrings/bookos.asc
