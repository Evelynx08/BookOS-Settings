BookOS APT

Debian and Ubuntu package repository for BookOS Settings

stable amd64 GPG signed

Installation

Step 1 — Import the signing key

sudo install -d -m 0755 /etc/apt/keyrings
curl -fsSL https://evelynx08.github.io/BookOS-Settings/KEY.gpg \
    | sudo tee /etc/apt/keyrings/bookos.asc > /dev/null

Step 2 — Register the repository

echo "deb [signed-by=/etc/apt/keyrings/bookos.asc] \
https://evelynx08.github.io/BookOS-Settings bookworm main" \
    | sudo tee /etc/apt/sources.list.d/bookos.list

Step 3 — Install the package

sudo apt update
sudo apt install bookos-settings
Compatibility. The bookworm codename is built against Debian 12 and is also compatible with Debian 13 and Ubuntu 22.04 or newer. Required runtime libraries: libwebkit2gtk-4.1-0 and libgtk-3-0.

Updates

New releases are delivered automatically through your normal system update workflow:

sudo apt update
sudo apt upgrade

Removal

To remove the package and unregister the repository:

sudo apt remove bookos-settings
sudo rm /etc/apt/sources.list.d/bookos.list
sudo rm /etc/apt/keyrings/bookos.asc
